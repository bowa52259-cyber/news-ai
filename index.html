📋 Google AI Studio 申請 API Key 步驟
1 點選上方連結前往 Google AI Studio（aistudio.google.com），使用 Google 帳號登入
2 登入後，點選左側選單的 「Get API key」 或頁面上方的 「API Keys」 按鈕
3 點選 「Create API key」，系統會自動建立一組 API Key
4 複製產生的 API Key（開頭為 AIzaSy…），貼到上方輸入欄位
✓ 免費額度每日可使用數百次，系統固定使用 Gemini 2.5 Flash 模型，效能與配額最佳
⚠️ API Key 請勿分享他人，Key 僅存於您的瀏覽器本地端，系統不會上傳或儲存於伺服器。

📘 填寫說明與範例
以下為常見的新聞稿類型與七段式結構說明
常見新聞稿類型
A
法規宣導型
說明法規內容、申報方式與計算方式
範例：房地合一稅優惠稅率申請條件說明
B
查核案例型
揭示違規情形，強調補稅罰鍰後果
範例：漏報租金所得遭補稅並裁罰
C
民眾詢問型
解答來電民眾常見稅務疑義
範例：繼承房地如何申報所得稅
D
節稅規劃提醒型
揭示節稅誤區與反面案例
範例：代繳土地增值稅被視同贈與
E
活動公告型
公告講習、申報期限等行政事務
範例：114年度租稅法規講習報名
